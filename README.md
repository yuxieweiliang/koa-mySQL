>  # 新增
向数据库中插入新数据: INSERT INTO

>  # 查询
从数据库中提取数据: SELECT * FROM test1

>  # 更新
更新数据库中的数据: UPDATE

>  # 删除
从数据库中删除数据: DELETE

>  # 床架一个数据库
创建新数据库: CREATE DATABASE

>  #
修改数据库: ALTER DATABASE

>  #
创建新表: CREATE TABLE

>  #
变更（改变）数据库表: ALTER TABLE

>  #
删除表: DROP TABLE

>  #
创建索引（搜索键）: CREATE INDEX

>  #
删除索引: DROP INDEX
